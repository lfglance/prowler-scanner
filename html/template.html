{{ team_name }} Security Scan

We offer a security scan using an open source tool, Prowler. To get started, follow the steps below:

1. Login as an administrator to the AWS account where the scan will be run.
2. Click the button below to launch a new CloudFormation stack.
3. Enter in a random alphanumeric string for the ExternalId parameter.
4. Click Create Stack at the bottom of the page.
5. Navigate to the Stack Outputs tab and provide {{ team_name }} with the RoleArn.
6. Coordinate a review call with {{ team_name }}; the scan will complete in the background.
